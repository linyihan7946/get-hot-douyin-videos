# 抖音热门视频与评论抓取工具

基于 Playwright 拦截抖音 Web 端 API 数据，实现获取指定课题热门视频并自动提取评论信息的工具。

## 功能介绍
1. **自动搜索**：根据指定关键词，搜索抖音视频。
2. **热度排序**：自动获取各视频的点赞量，并从高到低排序，过滤出最热门的视频链接。
3. **内容描述**：通过 API 提取视频文案/描述内容，展示视频大概讲解的内容。
4. **评论提取**：自动进入热门视频详情页，截获最新、最热的评论内容。
5. **JSON存储**：将所有采集到的结构化数据保存至本地 JSON 文件中。

## 环境依赖
该脚本依赖 Python 3.8+ 及 Playwright。在使用前请确保已正确安装依赖库。

### 安装步骤

1. 安装所需的 Python 包：
```bash
pip install playwright
```

2. 安装 Playwright 内置浏览器引擎（必须运行此命令）：
```bash
playwright install chromium
```

## 使用方法

由于抖音具备极其严格的反爬机制和弹出的滑块验证码，**脚本默认采用有头模式（可见浏览器窗口）启动**，方便用户在初次使用时或遇到验证码时手动处理（例如扫码登录或拖拽滑块）。

### 运行脚本示例

在终端中执行以下命令，`-k` 参数为必填，表示你要检索的课题名称；`-m` 参数为可选，表示要提取评论的最大视频数量（默认提取排名前 5 的视频）：

```bash
python douyin_spider.py -k "Python入门" -m 5
```

### 运行流程详解
1. 终端提示后，浏览器将自动弹出并加载抖音首页。
2. **手动操作阶段**：请在打开的浏览器窗口中，通过“扫码”或“手机验证码”登录自己的抖音账号。登录成功并在终端敲击 **`回车键`** 继续运行。（注意：如果遇到验证码弹窗也请手动处理）。
3. 脚本会自动跳转至该课题的搜索结果页，通过滚轮自动加载多页数据，提取所有检索到的视频及其点赞量、分享量和描述文本。
4. 随后，脚本会将这些视频按点赞量从高到低排序，挑选前 N 个最高点赞量的视频。
5. 脚本自动依次打开这些最热门的视频，并抓取前 50 条左右的评论信息。
6. 最终将结构化的数据导出至类似于 `douyin_result_Python入门.json` 的文件中。

## 注意事项
- 由于抖音的 API 加密签名（`X-Bogus` / `a_bogus` 等）更新频繁，本脚本不直接构造 HTTP 请求，而是使用 Playwright 网络拦截（Network Interception）功能捕捉浏览器自然发出的 API 响应。这是目前最稳定、安全的抓取思路。
- 频繁抓取可能导致当前 IP 被暂时封禁或强制要求扫码验证，请合理控制抓取频率和数量。
